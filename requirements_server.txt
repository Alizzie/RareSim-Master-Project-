# SETUP for server (first time only — skip if you already have the repo and venv):
#
#   1. SSH into server:
#      ssh username@anna.ifi.uzh.ch
#
#   2. Clone the project:
#      git clone https://github.com/Alizzie/RareSim-Master-Project-.git
#      cd RareSim-Master-Project-
#
#   3. Switch to new-architecture branch:
#      git checkout new-architecture
#
#   4. Create virtual environment:
#      python3 -m venv venv
#
#   5. Activate:
#      source venv/bin/activate
#
#   6. Install server dependencies:
#      pip install -r requirements_server.txt
#
#   7. Install project in editable mode:
#      pip install -e . --no-deps
#
# IF YOU ALREADY HAVE THE REPO:
#
#   git checkout new-architecture
#   git pull origin new-architecture
#   source venv/bin/activate
#   pip install -r requirements_server.txt
#   pip install -e . --no-deps
#
# ACTIVATE (every time you reconnect):
#
#   cd /home/user/your_username/RareSim-Master-Project-
#   source venv/bin/activate
#
# ============================================================
# HOW TO RUN — CHECK GPUs FIRST
# ============================================================
#
# Always check which GPUs are free before running:
#   nvidia-smi
#
# ============================================================
# HOW TO RUN — TRANSFORMER PIPELINE
# ============================================================
#
# Builds embedding cache for all 5 models (one-time, ~10 min on GPU).
# After first run, cache is reused — ranking is near-instant.
#
# For example if GPU 4 is available:
#   CUDA_VISIBLE_DEVICES=4 nohup python src/similarity_methods/transformer/pipeline.py \
#     > outputs/transformer/transformer_log.txt 2>&1 &
#
# Monitor:
#   tail -f outputs/transformer/transformer_log.txt
#
# Check if done:
#   ps aux | grep transformer/pipeline.py
#
# ============================================================
# HOW TO RUN — LLM PIPELINE
# ============================================================
#
# Runs Mistral-7B-Instruct-v0.2 for disease retrieval + explanation.
# Uses 4-bit quantization — fits in ~6GB GPU memory.
# Takes ~3-5 min per run (retrieval + explanation for 10 results).
#
# For example if GPU 4 and 5 are available:
#   CUDA_VISIBLE_DEVICES=4,5 nohup python packages/raresim-core/src/raresim/similarity_methods/llm/pipeline.py  \
#     > outputs/similarity_methods/llm/llm_log.txt 2>&1 &
#
# Monitor:
#   tail -f outputs/similarity_methods/llm/llm_log.txt 
#
